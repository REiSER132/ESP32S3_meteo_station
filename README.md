# ESP32S3_meteo_station
My home meteostation =)

## Hardware
- ESP32-S3 N16R8
- AHTx0 Sensor
- SSD1306 OLED Display
- BMP280 Pressure Sensor

## Libraries Used
This project uses the following open-source libraries:
- [AHTx0 Driver](https://github.com/adafruit/Adafruit_CircuitPython_AHTX0) by Adafruit Industries & Andreas Bühl
- [BMP280 Driver](https://github.com/davidstenwall/micropython-bmp280) by David Stenwall
- SSD1306 Driver (Standard MicroPython implementation)
